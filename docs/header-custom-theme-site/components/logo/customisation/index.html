Logo

StableAccessible

The logo is used to identify the site or application as an official City of Helsinki service.

  • Usage
  • Code
  • Accessibility
  • Customisation

Customisation

Logo customisation works differently!

Logo is an older HDS component and therefore its customisation works slightly differently. Instead of the theme property, you need to use the style property. This will be fixed in a future update.

Customisation properties

In Core version, you can either use the style or class attributes in the HTML to customise the component.

In React version, you can either use the style or className properties to customise the component.

Customisation example

  • React
Press Enter to start editing. Press Esc to stop editing.
import { Logo } from 'hds-react';


() => (
<>
  <Logo language="fi" size="small" style={{ color: 'var(--color-bus)', marginRight: 'var(--spacing-xs)' }} />
  <Logo language="fi" size="medium" style={{ color: 'var(--color-tram)', marginRight: 'var(--spacing-xs)' }} />
  <Logo language="fi" size="large" style={{ color: 'var(--color-metro)' }} />
</>)
